Hints

What is the use of tags?

Tags allow you to add multiple values.

How to create tags?

Type a value and press Enter to create a tag.

Which fields support tagging feature?

This feature is available for Programming Language, URL, Funding, License, Email and Keywords fields.

How to edit the tags?

Click on a tag to load it into the input field, edit it, and press Enter to update the tag.

How to delete a row from the table?

Select a row. The delete icon will appear, click and confirm to delete the row.

How to merge two rows in the table?

Select two rows with the same given and family name. The merge icon will appear, click and confirm to merge.

How to add a new row to the table?

Scroll to the end of the table, fill in the input fields, and press Enter or click the Add button to add a new row.

How to assign roles?

Click the Contributors, Authors, or Maintainer checkboxes to assign roles. The JSON will be updated accordingly.

How to see the explanation of contributors, authors, and maintainers?

Click the table header name (Contributors, Authors, or Maintainers) to check a detailed explanation.

How can the Related Person table be sorted?

The Related Person table can be sorted by Given Name and Family Name.

What does this small triangle indicate?

The triangle icon shows the sorting order: upward for ascending (A–Z, empty cells first) and downward for descending (Z–A).

How to know according to which column table is sorted?

The highlighted column header shows which column is currently being used for sorting.

Why do we have a JSON viewer?

The JSON viewer helps you to follow the changes which you made in the form.

What is the use of the toggle button?

The toggle button lets users choose whether they want to view the JSON viewer or hide it.
